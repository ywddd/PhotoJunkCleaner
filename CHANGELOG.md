# Changelog

## 1.2.0

### 结果页（相册式）
- 扫描完成后按分类显示：外卖、快递、二维码、支付、验证码、聊天、普通截图等
- 每个分类一张「相册封面」：封面缩略图 + 数量 + 副标题
- 点进分类后 **网格并排** 浏览（可 2/3/4 列）
- 支持分类内全选 / 清空、点选勾选、点图全屏预览与左右滑动
- 底部统一删除栏，删除前二次确认
